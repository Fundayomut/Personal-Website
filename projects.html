Rezept-Webanwendung

Technologien: React, React Router, React Bootstrap, Edamam API

Dieses Projekt ist eine Webanwendung, die verschiedene Rezepte und Nährwertinformationen bereitstellt. Benutzer können Rezepte für Frühstück, Mittagessen, Abendessen und Snacks durchsuchen. Die Anwendung bietet eine benutzerfreundliche Erfahrung mit dynamischen Komponenten und einer modernen Benutzeroberfläche. Rezepte und Nährwertinformationen werden über die Edamam API bereitgestellt.
